# 모바일 앱 배포 및 설치 가이드

현재 웹사이트는 **PWA (Progressive Web App)** 기술이 적용되어 있어, 별도의 앱 스토어 심사 없이도 스마트폰에 앱처럼 설치하여 사용할 수 있습니다.

## 1. 스마트폰에 앱으로 설치하는 방법

### 아이폰 (iOS)
1.  Safari 브라우저로 웹사이트에 접속합니다.
2.  화면 하단의 **공유 버튼** (네모에서 화살표가 위로 나가는 아이콘)을 누릅니다.
3.  메뉴를 아래로 스크롤하여 **'홈 화면에 추가'**를 선택합니다.
4.  오른쪽 상단의 **'추가'** 버튼을 누르면 홈 화면에 앱 아이콘이 생성됩니다.

### 안드로이드 (Android)
1.  Chrome 브라우저로 웹사이트에 접속합니다.
2.  화면 상단의 메뉴 버튼 (점 3개)을 누르거나, 화면 하단에 나타나는 **'홈 화면에 추가'** 배너를 누릅니다.
3.  **'앱 설치'** 또는 **'홈 화면에 추가'**를 선택합니다.
4.  설치가 완료되면 앱 서랍과 홈 화면에서 앱을 찾아 실행할 수 있습니다.

---

## 2. 앱 스토어(Play Store, App Store)에 정식 배포하려면?

현재 방식(PWA)으로도 충분히 앱처럼 쓸 수 있지만, 플레이 스토어나 앱 스토어에서 검색해서 다운로드받게 하고 싶다면 **네이티브 래퍼(Native Wrapper)** 작업이 필요합니다.

### 필요한 과정
1.  **Capacitor** 또는 **React Native WebView** 같은 도구를 사용하여 웹사이트를 감싸는 앱 프로젝트를 생성해야 합니다.
2.  **개발자 계정 등록**:
    *   Google Play Console: $25 (1회)
    *   Apple Developer Program: $99 (매년)
3.  **심사 및 배포**: 각 스토어의 가이드라인에 맞춰 심사를 받아야 합니다.

**추천**: 개인적인 용도나 소규모 그룹이 사용하는 경우, 현재의 PWA 방식이 비용과 관리 측면에서 가장 효율적입니다.
